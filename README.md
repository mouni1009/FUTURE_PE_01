# AI Website Copy Generator for Coaching Institutes

## Business Chosen
Raju Coaching Center, Gajuwaka, Visakhapatnam

## Problem
Many coaching institutes have unclear website content, weak messaging, and poor calls-to-action, which affects student admissions.

## Solution
I created a structured AI prompt system to generate:
- Homepage copy
- Services page content
- Strong CTA sections

## Tools Used
- ChatGPT

## Prompt Logic
The prompt includes:
- Business details (name, location, audience)
- Clear structure (homepage, services, CTA)
- Focus on benefits and trust

This ensures the output is practical and ready for real websites.

## Output
The generated website copy is available in the outputs folder.
